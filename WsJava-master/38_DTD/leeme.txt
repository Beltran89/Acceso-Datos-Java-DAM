Un DTD (Document Type Definition) es un archivo que define la estructura 
de un documento XML, es decir, el nombre de los elementos que contiene, 
sus atributos, el orden en el que tiene que aparecer cada elemento, 
si se repiten, y si pueden o no tener elementos hijos.

Para crear un fichero dtd
boton derecho -> new -> DTD

Para validar un documento xml que tenga un DTD asociado podemos pulsar en eclipse
boton derecho sobre el fichero xml -> validate
